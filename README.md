# Student Management System (CSV-Based)

A very simple Python console application to store and manage student records using a CSV file.  
This project focuses on **file handling**, **CSV operations**, and **basic CRUD functions**.
##  Features
- Add a student  
- Search for a student  
- Delete a student  
- Prevent duplicate student IDs  
- Stores data in `students.csv`  
##  Files
- `student_management.py` — Main script  
- `students.csv` — Automatically created with headers  
- `README.md` — Instructions
